# Tata Cara Pengerjaan Tugas

- Di dalam setiap folder, Anda akan menemukan berkas `README.md`. Bacalah dengan seksama untuk mengetahui tugas yang harus dikerjakan pada folder tersebut.
- Kerjakan setiap tugas yang diberikan dengan benar. Anda boleh menggunakan code editor apa pun dalam mengerjakan tugas.
- Di setiap folder, terdapat berkas validate.js yang dapat digunakan untuk menguji kebenaran tugas yang telah Anda kerjakan. Jika jawaban Anda benar, output-nya adalah passed!
